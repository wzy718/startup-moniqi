# 界面设计文档

> 本文档定义游戏所有界面（竖屏设计）
> 设计原则：紧凑布局、一屏显示、符号优先
> 屏幕基准：约36个中文字符宽度

---

## 1. 主界面（核心面板）

```
┌──────────────────────────────────┐
│ ☰  第24周·夏        💰¥128,500  │ ← 顶栏：周数+现金
├──────────────────────────────────┤
│ 😰 62 ████████░░  ❤️ 85 ████████░│ ← 压力/健康条
│ 👤 王小明 25岁 ENTJ  ⭐58  💑单身│ ← 人物一行
├──────────────────────────────────┤
│ 📊 本周: +¥3,200  💳 负债:¥5万  │ ← 本周净利+负债
├──────────────────────────────────┤
│ 🏪 我的店铺 (1)            [+新] │ ← 店铺区标题
│ ┌──────────────────────────────┐ │
│ │ ☕ 老王奶茶  ⭐4.2  📍商场核心 │ │ ← 店铺名+评分+位置
│ │ 💵 日¥2,800  📈 周利+¥4,500  │ │ ← 日营收+周利润
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│ 📬 本周事件                      │ ← 事件区标题
│ ┌──────────────────────────────┐ │
│ │ 🎭 员工闹矛盾                 │ │ ← 事件标题
│ │ 小李和小张因排班争执...  [>] │ │ ← 摘要+进入按钮
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│    🏠       📊       🏆       ⚙️  │ ← 底部导航
│   首页     数据     成就     设置 │
└──────────────────────────────────┘
```

### 元素说明

| 符号 | 含义 | 备注 |
|------|------|------|
| 😰 | 压力 | 数字+进度条，>80红色警告 |
| ❤️ | 健康 | 数字+进度条，<30红色警告 |
| ⭐ | 声望 | 直接显示数值，0-100 |
| 💑 | 婚恋 | 单身/恋爱/已婚/离异 |
| 📈 | 利润 | 正数绿色，负数红色 |
| 📬 | 事件 | 有事件时闪烁高亮 |
| [+新] | 开新店 | 点击进入开店流程 |
| [>] | 进入 | 点击进入事件详情 |

### 交互说明

- 点击「店铺卡片」→ 进入店铺详情界面
- 点击「事件卡片」→ 进入事件决策界面
- 点击「+新」→ 进入开店流程
- 点击「底部导航」→ 切换页面

## 2. 事件决策界面

```
┌──────────────────────────────────┐
│ ←  第24周·事件          🎭 日常  │ ← 返回+周数+事件类型
├──────────────────────────────────┤
│  📰 员工闹矛盾                   │ ← 事件标题
│  ────────────────────────────── │
│  小李和小张因排班问题争执。      │ ← 事件描述
│  矛盾已影响店里氛围，其他员      │   (3-4行)
│  工也开始站队……                  │
├──────────────────────────────────┤
│ A. 召集会议调解                  │ ← 选项A：标题
│    😰+5 │ 🎲70% │ ✨ENTJ+15%    │ ← 影响+成功率+MBTI加成
├──────────────────────────────────┤
│ B. 各打五十大板                  │ ← 选项B：标题
│    😰+2 │ 👥士气-10             │ ← 影响
├──────────────────────────────────┤
│ C. 开除小李                      │ ← 选项C：标题
│    💰-2000 │ 👥士气-5           │ ← 影响
├──────────────────────────────────┤
│ D. 置之不理          🔒声望≥70  │ ← 选项D：锁定+解锁条件
├──────────────────────────────────┤
│       [ 🎬 看广告揭示最优 ]      │ ← 广告入口
└──────────────────────────────────┘
```

### 元素说明

| 符号 | 含义 | 备注 |
|------|------|------|
| 🎭 | 事件类型 | 日常/危机/机遇/世界，不同颜色 |
| 😰+5 | 压力变化 | 正数红色，负数绿色 |
| 💰-2000 | 现金变化 | 正数绿色，负数红色 |
| 👥 | 员工士气 | 影响全店员工 |
| 🎲70% | 成功概率 | 有随机性的选项显示 |
| ✨ | MBTI加成 | 符合性格有额外加成 |
| 🔒 | 条件锁定 | 灰显，提示解锁条件 |
| 🎬 | 广告 | 看广告揭示最优选项 |

### 结果弹窗

```
┌────────────────────────────┐
│      ✅ 调解成功！          │ ← 结果标题(绿色)
│  双方握手言和，氛围恢复。  │ ← 结果描述
│  😰+5  👥+10  ⭐+3         │ ← 数值变化
│      [ 继 续 ]             │ ← 关闭按钮
└────────────────────────────┘
```

| 符号 | 含义 | 备注 |
|------|------|------|
| ✅ | 成功 | 绿色标题 |
| ❌ | 失败 | 红色标题 |


### 元素说明

| 符号 | 含义 | 备注 |
|------|------|------|
| 👥85人 | 日均客流 | 人数 |
| 💵¥28 | 客单价 | 元 |
| 📊58% | 毛利率 | 百分比 |
| 🧹88 | 卫生评分 | 0-100，<60红色警告 |
| ↑ | 上涨 | 绿色 |
| ↓ | 下跌 | 红色 |
| → | 持平 | 灰色 |
| 优/良/中/差 | 技能等级 | 优>良>中>差 |
| 😊78 | 士气值 | 0-100，<50黄色警告 |
| ⋯更多 | 折叠菜单 | 点击展开更多操作 |

### 折叠操作（点击「⋯更多」展开）

| 按钮 | 功能 |
|------|------|
| 📦外卖 | 外卖平台开关、抽成、推广 |
| 🎨装修 | 花钱提升店铺环境 |
| 📢营销 | 设置推广预算 |
| 📋菜单 | 调整菜品 |
| 🏷️定价 | 调整客单价 |
| 🚪关店 | 暂停或永久关店 |

## 16. 设计原则

1. **一屏优先** - 核心信息无需滚动
2. **符号代文** - 用图标减少文字
3. **数值突出** - 关键数字加粗放大
4. **折叠次要** - 详情按需展开
5. **颜色区分** - 红负绿正灰中性黄警告
6. **反馈即时** - 操作有动画有振动
7. **阈值警告** - 关键数值超阈值变色
